ReScatter: Reactive Scatter plots library examples

Version 0.1

Introduction

The purpose of the ReScatter library is the quick production of sites containing very large (10^5 point) scatter plots which permit plot linking and selection brushing and the display of density maps. Selections can be exported for further analysis. The WebGL based plots and the ReScatter technology were first demonstrated on the Brainscope website.

History and usage

Brainscope framework refactored to ReScatter

On the original Brainscope site the framework code was not minified or uglified in any way. As such anyone using the site could potentially reuse the code as supplied. However the framework, while divided into a number of classes, lacked a clear structure. In order to facilitate open source reuse the framework has been refactored and renamed ReScatter.

Usage and limitations

In its current form the library is still quite restricted in its applicability but if your need plots like those in the Brainscope site that much is possible. The first release has deliberately been given a low version number (0.1) to indicate that much more refactoring is needed to permit the easy reuse of key individual components such as the ReScatter.view.PointGroupView that forms the core of the WebGL based plots.

Note on tSNE dual view

A tSNE dual-view is a technique of creating a tSNE embedding on high dimensionality data over both the sample and data dimensions. The result is samples grouped according to the data dimensions and the data groups according to the samples. By linking the plots the user can discover which data points make the greatest contribution to the sample clusters
